# Лабораторная работа №1 | Вариант 33
## Перевод десятичного числа в заданную систему счисления

### Описание
Программа переводит целое десятичное число (в том числе отрицательное) в систему счисления с основанием от 2 до 9. Перевод реализован вручную через алгоритм последовательного деления с остатком — без использования встроенных функций и импортов.

### Использование
1. Запустите программу:
```bash
python 1lab.py
```

2. Введите целое число (например: `34`)

3. Введите основание системы счисления от 2 до 9 (например: `7`)

4. Получите результат

### Пример работы
```
Введите целое число: 34
Введите систему счисления (2-9): 7

--- Результат ---
Число 34 в системе счисления 7 = 46
```

### Особенности программы
- **Без импортов**: алгоритм реализован с нуля
- **Отрицательные числа**: корректно обрабатываются
- **Обработка ошибок**: программа не падает при некорректном вводе
- **Graceful exit**: корректная обработка Ctrl+C и Ctrl+Z

### Алгоритм перевода
Число последовательно делится на основание, остатки от деления записываются снизу вверх:
```
34 ÷ 7 = 4, остаток 6
 4 ÷ 7 = 0, остаток 4
Результат (снизу вверх): 46
```
